# Este es el primer archivo
En este repositorio vamos a poner tambien algunos scripts de micropython para usarr el raspberry pi pico
El archivo dallas2sesores.py es un script para poder leer en raspberry pi pico dos sensores de temperatura tipo d1820. debe estar conectado a la pc pero me ha sido muy util para hacer experimentos simples como ver el impacto de la sal en la temperatura cuando se aplica en hielo. O ver el aumento de la temperatura cuando se aplica agua al detergente. Lastima que solo puedo verlo y mi proximo es agregar un ds3231 y un microSD
